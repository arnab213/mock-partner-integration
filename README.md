# Mock Partner Integration Sandbox
## About
This project is a personal side project to create a mock partner integration sandbox for testing purposes. It allows me to test and validate different API integrations with partners in a controlled environment.
## Tech Stack
* Java 17
* Spring Boot 3.x
* MySQL
* Maven
## Prerequisites
* Java 17 installed
* MySQL installed
* Maven installed
## Setup Steps
1. Clone the repository
2. Run `mvn clean install` to build the project
3. Create a MySQL database and update the `application.properties` file with the database credentials
4. Run `mvn spring-boot:run` to start the application
## Usage Examples
* Use Postman or cURL to send requests to the API endpoints
* Test different API integrations with partners
## Folder Structure
* `src/main/java`: Java source code
* `src/main/resources`: application properties and configuration files
* `pom.xml`: Maven project file
* `docs`: documentation and architecture notes
